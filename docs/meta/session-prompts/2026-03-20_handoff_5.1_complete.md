## Decisiones de Diseño (Bloque 5.1 — definitivas)

1. **QuickInstruct = siempre KnowledgeItem**: Todo input del dueño (incluyendo PDFs ricos) se destila en un KnowledgeItem. El LLM extrae la esencia relevante, no transcribe el documento.
2. **Panel expandido durante procesamiento**: UX crítico — el panel no colapsa hasta que el LLM responde. El dueño ve su input mientras espera.
3. **Chip de archivo inmediato**: El nombre del archivo aparece con spinner desde el instante de selección. Submit bloqueado hasta que base64 esté listo.
4. **PDF → Gemini obligatorio**: `application/pdf` como `image_url` no funciona en modelos Azure/OpenAI. Solo Gemini lo soporta vía OpenRouter.
5. **Strip de markdown fences es obligatorio**: Gemini puede devolver `\`\`\`json...\`\`\`` en vez de JSON puro. Siempre sanitizar antes de `JSON.parse`.
