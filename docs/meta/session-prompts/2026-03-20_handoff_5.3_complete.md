### Autonomía MVP — implementación simplificada
```typescript
// Si CUALQUIER autonomy_rule del negocio está activa + confidence = high → auto_sent
const { data: autonomyRules } = await supabase
    .from('autonomy_rules')
    .select('id')
    .eq('business_id', business.id)
    .eq('active', true)
    .eq('level', 'autonomous_with_guardrails')
    .limit(1)
const shouldAutoSend = (autonomyRules?.length ?? 0) > 0 && confidenceTier === 'high'
```
